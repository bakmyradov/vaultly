=============================================================================
KEY FEATURES IMPLEMENTED
=============================================================================

1. MULTI-METHOD QR INPUT
   - Upload QR code from image files
   - Real-time camera scanning with jsqr detection
   - Manual text/URL entry with QR generation

2. VAULT MANAGEMENT
   - Add, edit, delete passes with localStorage persistence
   - Category organization (Gym, Transit, Loyalty, Work, Travel, Other)
   - Search and filter by name and category
   - Pass metadata: name, category, notes, QR data, timestamps

3. PASS DISPLAY
   - 2-column responsive grid layout
   - Individual pass detail view
   - Fullscreen QR display (for scanners at checkout)
   - Wake lock + fullscreen APIs for scanner usage

4. PWA CAPABILITIES
   - Installable to home screen
   - Offline-first with Workbox caching
   - Auto-updating service worker
   - Custom splash screens and theme colors
   - Works on mobile and desktop

5. UI/UX
   - Tailwind CSS (CDN-based, no build step needed)
   - Custom color scheme: Primary #185FA5, Dark #0C447C, Light #E6F1FB
   - 12px rounded corners on cards, 8px on inputs
   - System fonts for native feel
   - Responsive design (mobile-first)
